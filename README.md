# CryptoVerse

## 🚀 Overview

CryptoVerse is a beginner-friendly cryptocurrency education platform designed to simplify blockchain and cryptocurrency concepts through interactive learning.

Users can learn about Bitcoin, Ethereum, Blockchain, Web3, Crypto Wallets, and Crypto Security in an engaging and easy-to-understand way.

## ✨ Features

* Cryptocurrency Basics
* Blockchain Fundamentals
* Bitcoin vs Ethereum Comparison
* Crypto Wallet Guide
* Crypto Security Awareness
* Interactive Quiz System
* Live Cryptocurrency Prices
* Responsive Design
