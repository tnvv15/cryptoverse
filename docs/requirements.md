# CryptoVerse Requirements

## Functional Requirements

1. Users can learn cryptocurrency concepts.
2. Users can learn blockchain fundamentals.
3. Users can compare Bitcoin and Ethereum.
4. Users can attempt quizzes.
5. Users can view live cryptocurrency prices.
6. Users can learn crypto security practices.

## Non-Functional Requirements

* Responsive Design
* Fast Performance
* User-Friendly Interface
* Mobile Compatibility
* Modern UI/UX

## Target Audience

* Students
* Beginners
* Crypto Enthusiasts
* Blockchain Learners
